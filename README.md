## 📌 Project Overview
Analyzed **1,80,519 global supply chain records**
to identify delivery risks, profitability
drivers, and operational inefficiencies
across 5 global markets.

## 🗂️ Dataset
| Detail | Info |
|--------|------|
| Source | DataCo Smart Supply Chain — Kaggle |
| Rows | 1,80,519 |
| Columns | 43 |
| Markets | LATAM, Europe, Pacific Asia, USCA, Africa |

## 🛠️ Tools Used
| Tool | Purpose |
|------|---------|
| MySQL | 12 Business SQL Queries |
| Python | EDA + Risk Analysis |
| Power BI | 1-Page Dashboard |

## 📊 Key Business Insights
1. Late delivery rate: **54.8%** — Critical issue
2. **First Class shipping = 95.3% late rate**
   *(Counterintuitive key finding!)*
3. Discounts above 20% = **consistent losses**
4. Fan Shop = highest profit department
5. Europe = highest revenue market ($9.77M)
